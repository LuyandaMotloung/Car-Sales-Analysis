# Car-Sales-Analysis
## Project Overview
This project analyses car sales data to uncover sales trends and patterns across different car brands and models. This project identifies top-perfoming brands, regional patterns and customer preferences. The insights from this project are designed to support data driven decisions in marketing and sales strategies, as well as inventory management. 

## Tools Used
- Microsoft Excel: Data cleaning and processing
- Power BI: DAX measures, interactive dashboard and data visualisation

## Dataset
- Source: Exodus Experts
- Size: 3 worksheets in the file
  i.e. Dealers: 10 records, 4 columns; Models: 15 records, 8 columns; Sales: 218 records, 7 columns

## Analysis Steps
1. Data import: Loaded the car data excel file.
1. Data cleaning and transformation: Corrected data types, removed blank rows
2. Data modeling: Created relationships between the tables.
3. Exploratory data analysis and visualisation: Utilised DAX measures to define key metrics and built an interactive dashboard using charts and slicers.
4. Key insights: Identified top-perfoming brands, regional patterns and customer preferences.

## Key Insights
1. BMW and Mercedes Benz consinstently generate the highest revenue, specifically the 3 series and the GLA-Class.
2. Most customers prefer Sedans, they account for the largest proportion of total car sales.
3. Gaborone, a city in Botswana accounted for the most sales, reflecting a strong urban demand.

## Recommendations
1. Optimise inventory to ensure that the top-perfoming brands are always in stock, this will help maximise revenue.
2. Adjust marketing strategies and stock based on regional and customer preferences.
